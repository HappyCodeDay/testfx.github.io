TestFX
Simple and clean testing for JavaFX

3.1.2
Feb 1, 2014

Features

A ﬂuent and ﬁxture-less API.

Robots to interact with JavaFX.

Matchers to verify behaviour with Hamcrest.

Services to launch applications in JUnit, TestNG and Spock, setup and lookup stages and controls.

Examples
JUnit 4TestNGSpock (in Groovy)

class DesktopTest extends GuiTest {
    public Parent getRootNode() {
        return new Desktop()
    }

    @Test
    public void should_be_able_to_drag_file_to_trashcan() {
        // given: "a file was created on an empty #desktop"
        rightClickOn("#desktop").moveTo("New").clickOn("Text Document");
        type("myTextfile.txt").push(ENTER);

        // when: "the file is dragged into #trashcan"
        drag(".file").dropTo("#trashcan");

        // then: "#desktop contains zero files"
        verifyThat("#desktop", contains(0, ".file"));
    }
}

Compatibility

Supports JavaFX 2 and JavaFX 8.
Supports Hamcrest matchers with lambda expressions.
Supports JUnit, TestNG, Spock (Groovy)
